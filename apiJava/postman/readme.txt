You can load this file in your postman and test it.

1- Install Postman application (https://www.getpostman.com/)
2- Start Postman, connect with your Gmail account or what ever
3- Next to the 'New' button, click the 'Import' button and select the Stone Lunchtime.postman_collection.json
4- You should see the collection at the left of your sreen
5- Start the back of the Stone Lunchtime Java Project (with command line or Eclipse)
6- Start the postman collection clicking the arrow next to the collection, and Run
